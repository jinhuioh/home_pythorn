자바의 컬렉션(자료구조)
-----------
-순서:List, ArrayList
-중복x:Set,HashSet
-FIFO: First Input First Output(선입선출)
  예시) 대기줄(queue,큐 구조)구조,LinkedList
       쇼핑몰 등에서 최근 본 목록
-FIFLO: 선입후출
-key+value: Map, HashMap



파이썬의 컬렉션
------------
-list(): 순서 중요, [ ]
-set(): 중복x, { }
-tuple(): 순서, 읽기 전용, ( )
-dict() : key+value, { }, dictionary구조, 사전
